# CreditRunner

## 概要
Kinectで遊べるUnity製3Dスクロールアクションゲームです。  
研究室体験の授業で自主制作したときの作品です。  
Kinectに骨格情報を読み込ませ、身体を動かして画面上のキャラクターを操作します。  
まだ未完成ですが、実際のゲームのシステムを改良中です。  

## 開発環境  
* Unity 2017.2.0f3
* Visual Studio Community 2015  
    Unity上から自動で呼び出せるように設定してエディタとして使いました。
